# Belajar Programming Interaktif

Minimal Vite + React setup untuk aplikasi quiz `pretest-webdev.jsx`.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

Vercel mendeteksi project Vite secara otomatis. Pastikan branch `main` sudah berisi `package.json`, `vite.config.js`, dan `index.html`.
